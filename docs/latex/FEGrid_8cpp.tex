\doxysubsection{Detailed Description}
Implementation of the \doxylink{classFEGrid}{FEGrid} class for managing finite element grid data. 

This file contains methods for handling the grid\textquotesingle{}s nodes and elements, including methods for calculating gradients of shape functions, element areas, and accessing node and element data.
